## 小哒 Coda v1.2.0

「回车键」团队 · Coding Agent 桌宠 · 基于 clawd-on-desk 二开。

### ✨ 本次更新

**桌宠全状态换新画风 + 色系统一**
- 10 个动画状态（idle / waking / thinking / working / juggling / sweeping / happy / notification / error / sleeping）全部换成新画风的多帧循环动画，告别单调静态图。
- **色系统一**：把所有状态的主体橙色对齐到同一基准（暖黄橙），消除此前深浅不一、个别状态偏深橘/偏淡的问题。全状态色相跨度收敛到 0.4° 以内。
- **顶部留白修复**：修正高举手臂 / 头顶元素（灯泡、抛球、问号）被窗口边缘裁切的问题，所有状态统一保留顶部呼吸空间。
- idle 挥手状态修复：举起的手臂与身体同色，不再突兀深橘。

**新功能：一键展示所有状态**
- 右键菜单 / 托盘菜单新增「一键展示所有状态」，自动依次播放全部桌宠动画，再次点击可中断。支持中 / 繁 / 英 / 韩 / 日五语言。
